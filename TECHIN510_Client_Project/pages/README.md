# 📱 Application Pages Documentation

## 1_Media_and_Mood.py

This is the main page of the multimedia journal application, integrating the following features:

### 🎨 Mood Tracking
- Supports 6 different mood states
- Real-time display of current date and selected mood

### 📸 Media Upload
- Supports images (PNG, JPG, JPEG)
- Supports videos (MP4)
- Supports audio (MP3)
- File size limit: 200MB
- Support for adding text descriptions to media

### 📝 Journal Features
- Large text editing area
- Auto-save functionality
- Emoji support

### 📊 History Records
- Displays 3 most recent media uploads
- Displays 5 most recent journal entries
- Sorted by date in descending order

### 💾 Data Storage
- Uses SQLite database
- Automatic table creation
- Local persistent storage
